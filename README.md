# BridgeUp — Human Needs Operating System

An AI-powered platform that matches people in need with verified helpers,
resources, jobs, and training in real time. Like Uber, but for solving
poverty, unemployment, and resource gaps globally.

## Vision
Post your need. Get matched. Get help. No hassle.

## Target Communities
- Underserved communities in the USA, Canada, Rwanda, and across Africa
- Works on smartphones AND feature phones (SMS/USSD)
- Available in multiple languages, including Kinyarwanda

## Tech Stack
- Frontend: HTML, CSS, JavaScript (PWA — installable on iPhone)
- Backend: Node.js + Express
- Database: Firebase Firestore (Google Cloud)
- AI: Claude API (Anthropic) — real dynamic conversations
- Maps: Leaflet.js + CartoDB Positron tiles (Uber-style full-screen maps)
- SMS: Twilio (OTP + feature phone support)
- Payments: Stripe (global) + Africa's Talking (MTN, Airtel, Tigo — Africa)
- Charts: Chart.js
- Hosting: Replit (dev) → Railway + Firebase Hosting (production)

## Key Features
- Uber-style full-screen live map with animated helper markers
- AI-powered need intake with Claude API
- Real-time GPS tracking and distance calculation
- Two-way review and trust system
- 8 role-based dashboards (Admin, Donor, Employer, NGO, Helper, and more)
- Automated impact reports (PDF + Excel)
- MTN Mobile Money + Airtel Money for Rwanda and Africa
- Feature phone SMS support for users without internet
- White-label multi-tenant architecture
